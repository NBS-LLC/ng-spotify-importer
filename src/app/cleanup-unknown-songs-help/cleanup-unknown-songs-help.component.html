Help - Cleanup Unknown Songs

Cleanup Unknown Songs is an experimental feature. When used it removes special characters from unknown songs.

The idea being that the "cleaned up" songs will then be found (known).

The following cleanup occurs on each unknown song's title and artist:

  • the parenthesis portion is removed
  • the ampersand (&) portion is removed
  • single quotes (') are removed
  • the "featuring" portion is removed
